unstack.vim
=============

Open stacktraces in vim splits!

Usage
-----
Visually select part/all of a stacktrace and hit `<leader>s`
 If you want to use a different map key, put `let g:unstack_mapkey=<F10>` (replacing <F10> with the key of your choice.)
